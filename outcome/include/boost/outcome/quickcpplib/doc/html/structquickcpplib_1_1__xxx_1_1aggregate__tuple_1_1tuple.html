template<class... Args>
struct quickcpplib::_xxx::aggregate_tuple::tuple< Args >

A C++ 14 aggregate initialisable collection of heterogeneous types standing in for C++ 17 tuple.

Template Parameters
ArgsAn arbitrary list of types

This is a highly incomplete wrapper of std::tuple. It has one major feature the C++ 14 std::tuple does not:

constexpr tuple<int, double, const char *> fc[] = {{1, 1.0, "hi1"}, {2, 2.0, "hi2"}, ...};
